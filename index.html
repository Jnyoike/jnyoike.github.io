About Me

Hey there! I am a Computer Science and Mathematics student at Smith College, an aspiring software engineer. My interests include front-end development, teaching and mentoring. So far, I have worked with technologies such as iOS and Android. When not coding or studying, I like travelling and finding new ways to give back to the community that has shaped me into who I am.

Projects

RateMyClass

During my J-Term workshop with Google, my group created an app that allows Smith students to rate and give feedback on the classes that they have taken. The feedback is then used by other students to get an idea on how their semester would be like, for instance how many hours outside of class would they need to work on the material for the class and how often do they get quizzes, if they took the class with a cetain professor and thus give them an opportunity to balance out their work. We are hoping to deploy it onto the play store really soon.

Voice Action Builder

Built a xml voice builder for the Africa's Talking Python SDK for use with their Voice API. It enables developers to use the voice API without hardcoding the xml strings required to use the API. They simply call different functions and the string is generated.

Africa's Talking Python SDK Example WebApp

During my sophomore year summer, I interned at Africa's Talking where I collaborated with the UI/UX team to build a simple example webApp for the Python SDK using Flask, JavaScript and RESTful APIs. It provides developers a simple template on how a few of the APIs work and how to use them.
